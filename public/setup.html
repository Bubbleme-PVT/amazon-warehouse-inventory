f
Meta Ads API
Connect Facebook & Instagram ad accounts
How to get your Meta credentials:
1. Go to developers.facebook.com → Create App → Business type
2. Add products: Marketing API
3. Permissions needed: ads_read + business_management + ads_management
4. Get long-lived token via Graph API Explorer
5. Ad Account ID: Business Manager → Ad Accounts (format: act_XXXXXXXX)

G
Google Ads API
Connect Search, Shopping, Display & YouTube campaigns
How to get your Google Ads credentials:
1. Google Cloud Console → New Project → Enable Google Ads API
2. Create OAuth 2.0 credentials → Web Application → add https://developers.google.com/oauthplayground as redirect URI
3. Get Developer Token: Google Ads UI → Tools & Settings → API Center (apply for Basic access)
4. Get Refresh Token: OAuth Playground → Step 1: select Google Ads API v14 → Authorize → Exchange for tokens
5. Customer ID: Google Ads → top-right (format: 123-456-7890)

Deploy to Production
Make AdSignal live on the internet — free in 5 minutes

Choose Railway (recommended) or Render. Both are free, support SQLite persistence, and give you a live HTTPS URL.

Option A — Railway Recommended
  1. Install Railway CLI: npm i -g @railway/cli
  2. Login: railway login
  3. From your adsignal/ folder, run:
railway init && railway up
  1. In Railway dashboard → Variables, add:
    JWT_SECRET = any random 32-char string
    PORT = 3001
  2. Add a Volume: Settings → Volumes → Mount at /app (persists your SQLite DB)
  3. Your live URL appears in the Railway dashboard — share it!
Option B — Render
  1. Push your adsignal/ folder to a GitHub repo
  2. Go to render.com → New → Web Service → connect your repo
  3. Build command: npm install | Start command: node server.js
  4. Add environment variable: JWT_SECRET = random string
  5. Add a Disk: /data mount (update DB_PATH in db/database.js to /data/adsignal.db)
  6. Click Deploy — live in ~2 minutes
After deploying: Open your live URL → log in → go to /setup → add your real Meta & Google API keys → click Sync Now. Your real campaign data will populate within 60 seconds.
